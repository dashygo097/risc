## Quick Start
just add 
```javascript
import { WaveDrom } from '@components'
```
at the top of a new `.mdx` file to use the plugin.

## Draw Waveforms
`WaveDrom` is a simple and effective way to create timing diagrams using `JSON-like` syntax,
and I wrote a plugin in `MDX` to render `WaveDrom` diagrams **directly in markdown files**. <br />
- you can use the waveform code block to create diagrams. <br />

## WaveDrom Examples

Here's a simple clock signal: 

```json
{ "signal" : [{ "name": "Alfa", "wave": "01.zx=ud.23.456789" }] }
```


```wavedrom
{ "signal" : [{ "name": "Alfa", "wave": "01.zx=ud.23.456789" }] }
```

```json
{ "signal": [{ "name": "clk",  "wave": "P......" }] }
```

```wavedrom
{ "signal": [{ "name": "clk",  "wave": "P......" }] }
```

```json
{ "signal": [
  { "name": "clk",         "wave": "p.....|..." },
  { "name": "Data",        "wave": "x.345x|=.x", "data": ["head", "body", "tail", "data"] },
  { "name": "Request",     "wave": "0.1..0|1.0" },
  {},
  { "name": "Acknowledge", "wave": "1.....|01." }
]}
```

```wavedrom
{ "signal": [
  { "name": "clk",         "wave": "p.....|..." },
  { "name": "Data",        "wave": "x.345x|=.x", "data": ["head", "body", "tail", "data"] },
  { "name": "Request",     "wave": "0.1..0|1.0" },
  {},
  { "name": "Acknowledge", "wave": "1.....|01." }
]}
```
