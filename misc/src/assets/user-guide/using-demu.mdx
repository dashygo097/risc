# DEMU Simulator

```info
This project is still under active development. 
```

`demu` is a **RTL-level** cpu and system simulator built with **Verilator**.

## Prerequisites

- **Make(Ninja)/CMake**
- **[spdlog](https://github.com/gabime/spdlog.git)**
- **[Verilator](https://github.com/verilator/verilator.git)**
- **lit(optional)**

## Building the Simulator

### Quick Build

```bash
make config # generates a `config.cmake` under `build/` for user to customize
make    # uses `Ninja` by default
```

### Manually Build

```bash
make config # You can edit `build/config.cmake` to customize the build, e.g. enable ccache, set the target architecture, etc.
mkdir -p build
cd build
cmake .. -G Ninja
ninja
```
